使用Yeoman搭建项目

所需软件

  • nodejs
  • git

安装Yeoman

Yeoman是一个含有插件系统的开源工具，可以使用最佳实践构建新的项目，搭配Grunt和Bower可以使开发更加高效；

Grunt是一个javascript的任务运行器，可以帮助完成自动构建和测试应用的重复性任务；

Bower是一个依赖管理工具，不用自己手动下载依赖包了，类似于微软的nuget;

使用npm安装以上工具

  • npm install -g grunt-cli
  • npm install -g bower
  • npm install -g generator-angular@0.9.8

使用npm install时指定-g 标识将保证安装包在机器中全局可用。
angularJs生成器的升级 npm install -g generator-angular

搭建项目

  1. 创建项目目录
  2. 进入该目录下 执行 yo angular 项目名称

运行项目

  1. grunt serve
  2. 若出现 Error: Cannot find where you keep your Bower packages. 问题，则运行bower install
  3. 若使用cmd命令行运行bower install 可能会提示 Bower : ENOGIT git is not installed or not in the PATH，这个时候可以将git加入环境变量path,也可以使用gitbash来代替cmd运行bower install命令就不会出现错误提示了，最后运行 grunt serve 浏览器会自动打开 http://localhost:9000 ,此时项目已经正常运行成功了。
